# Configure the VM-Series Plugin on the Firewall



 



 Select DeviceVM-Series to configure the
plugin integration for the cloud provider on which this instance
of the VM-Series firewall is deployed.

 
 
 
 
 
 
 

 

 
 
 
 

 

 

 

 If your firewall
is deployed on a hypervisor or cloud without a public interface
(for example, VMware ESXi), the tab is named VM-Series and displays
a general message.
