# License the VM-Series Firewall



 



 Learn about licensing for flexible licensing (Software
NGFW credits) and fixed vCPU model licenses.



 VM-Series firewall supports two license types (BYOL
and PayGo), and two different licensing models—Software Next Generation
Firewall Credits (Software NGFW) for flexible configurations that
you specify with a deployment profile, and fixed VM-Series Model
configurations. Both models also license Security services and other
features.

If you are an authorized CSSP partner, see [Licenses
for Cloud Security Service Providers (CSSPs)](/content/techdocs/en_US/vm-series/11-1/vm-series-deployment/license-the-vm-series-firewall/licenses-for-cloud-security-service-providers-cssps.html#id347ce605-cf75-4d8e-80c8-20dfb2242cbb) for information
that pertains to you.
