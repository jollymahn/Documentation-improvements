## Language Support



 
 The Strata Cloud Manager web interface supports localization. In addition to
 **English**, these are the languages that Strata Cloud Manager
 supports:

 
- **Chinese Simplified** (zh-cn)

- **Chinese Traditional** (zh-tw)

- **Spanish** (es-es)

- **Japanese** (ja-jp)

- **French** (fr-fr)

- **German** (de-de)
